# シラバスフォーマット仕様

## 文字数制限（条件式より）

### 授業概要
```
有効範囲: 51〜149文字 または 251文字以上
条件式: =OR(((LEN()>50)*(LEN()<150)),LEN()>250)
```

### 到達目標
```
有効範囲: 61〜99文字 または 251文字以上
条件式: =OR(((LEN()>60)*(LEN()<100)),LEN()>250)
```

### キーワード
```
3〜5個、各10文字以内（最大20文字）、日本語、商標不可
```

### 講義計画

| 項目 | 文字数制限 | 備考 |
|------|-----------|------|
| タイトル | 20文字以内 | 毎回異なる、句読点・記号不可 |
| 内容 | 150文字以内 | 毎回異なる、講義を含むこと |

## その他の項目

### 授業形式
```
「講義形式」または「講義および演習形式」のいずれか
```

### 成績評価
```
- 2要素以上
- 各要素の割合を%で明記
- 合計100%
- 単一要素60%以下
- 出席のみの評価不可
```

### 履修条件
```
前提条件があれば記載、なければ「なし」
```

### 教科書
```
書籍タイトル／著者／出版社／ISBN
なければ「なし」
```

### 参考文献
```
書籍タイトル／著者／出版社／ISBN またはURL
なければ「特になし」
```
